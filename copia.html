¿Qué es el CSS?

CSS, o hojas de estilo en cascada, es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML.

Es esencial para la creación de sitios web modernos y define cómo los elementos HTML deben ser visualizados en pantalla.

El origen del CSS

El CSS fue propuesto por primera vez por Håkon Wium Lie en 1994. Desde entonces, ha evolucionado hasta convertirse en uno de los pilares del desarrollo web.

¿Cómo funciona el CSS?

Existen varias maneras de aplicar CSS a un documento HTML, entre las que se incluyen:

Inline CSS

El CSS en línea se aplica directamente a los elementos HTML usando el atributo style dentro de las etiquetas.

Internal CSS

El CSS interno se escribe dentro de la sección <style> del documento HTML, generalmente en la cabecera.

External CSS

El CSS externo se enlaza a través de un archivo separado con la extensión .css, lo que facilita su mantenimiento y reutilización en múltiples páginas.

Selectores en CSS

Los selectores son patrones que se utilizan para seleccionar los elementos HTML que quieres estilizar. Hay varios tipos de selectores:

  • Selector de etiquetas
  • Selector de clases
  • Selector de ID
  • Selector de atributos
